# typeorm-sample Project Build with TypeORM

## 概要

TypeORM の動作サンプルです。

* `sample/` 以下 - [typeorm/sample at master · typeorm/typeorm](https://github.com/typeorm/typeorm/tree/master/sample)のコードをコピーの上、動作しなくなっているコードやMySQL非対応のコードを修正した内容
* `test/` 以下 - 独自に追加したテストコード

## Fork from typeorm/typeorm/sample/ repository

Steps to run this project:

1. Run `npm i` command
2. Run `npm run serve:db` command
3. Run `npm start:sample1` 〜 `npm start:sample33` command
